# Mondelbrot

Небольшая утилита на Go: рендерит анимированное приближение к множеству Мандельброта и сохраняет результат в `mandelbrot.gif`. Используется сглаженная оценка итераций (escape-time), параллельная отрисовка по строкам и палитра, подходящая для GIF.

![Множество Мандельброта (иллюстрация, Wikimedia Commons)](https://upload.wikimedia.org/wikipedia/commons/thumb/2/21/Mandel_zoom_00_mandelbrot_set.jpg/500px-Mandel_zoom_00_mandelbrot_set.jpg)

*Изображение: [Wolfgang Beyer](https://commons.wikimedia.org/wiki/File:Mandel_zoom_00_mandelbrot_set.jpg), CC BY-SA 3.0.*

## Требования

- [Go](https://go.dev/dl/) 1.24 или новее

## Запуск

```bash
go run .
```

После выполнения в текущей директории появится файл `mandelbrot.gif`.

Параметры анимации (разрешение, число кадров, центр, скорость зума) задаются константами в `main.go`.

## Структура проекта

| Файл        | Назначение                          |
| ----------- | ----------------------------------- |
| `main.go`   | расчёт Мандельброта и кодирование GIF |
| `go.mod`    | модуль Go                           |
